Other causes of elevated creatinine:
+ High intake of cooked meats, creatine supplements, high intensity excercise (muscle breakdown ==> creatinine formation)
+ Some drugs: NAC, cephalosporin, haemolysis
+ H2 blockers (cimetidine, ranitidine)

Theoretically, clearance should be measured using a substance which is freely filtered, not secreted and not reabsorbed. The ideal substance is inulin.
In practice creatinine is used because it is an endogenous substance. But creatinine is secreted in small amounts by the tubules (15% is secreted). Therefore, using creatinine as the molecule slightly overestimates the GFR.
Accurately measuring creatinine clearance would require measuring urine creatinine. So for convenience, we measure the serum creatinine and approximate the GFR based on that.

This is done using fancy formulas such as MDRD formula or the newer CKD-EPI formula which account for age, sex, race etc when producing an eGFR value. Source
